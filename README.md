Chess in Python using Pygame

# Build and Run

1. Windows - `py -m pip install -r .\requirements.txt`, Unix/macOS - `python -m pip install -r .\requirements.txt`
2. Windows - `py .\main.py`, Unix/macOS - `python .\main.py`
